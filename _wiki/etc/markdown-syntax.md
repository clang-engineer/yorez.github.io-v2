# 마크다운 문법

## code block
### 특정 언어를 명시하면 하이라이팅
``` (언어명시)
int val = 10;
printf(%s,"Hello, World!");
```

## 강조
### 기울여쓰기
- \* or \_
- ex> \*test\* or \_test\_
 
### 굵게 쓰기
- \*\* or \_\_
- ex> \*\*test\*\* or \_\_test\_\_

### 기울여 +  굵게 쓰기
- \*\*\* or \_\_\_
